# SpendTrack — Project Steering

## Stack
- HTML, CSS, Vanilla JavaScript only (no frameworks)
- LocalStorage for data persistence
- Chart.js for visualizations

## Folder Rules
- Only 1 CSS file inside `css/`
- Only 1 JavaScript file inside `js/`
- No backend, no build tools, no test setup

## Code Style
- Keep code clean and readable with section comments
- Validate all user inputs before processing
- Escape HTML output to prevent XSS
